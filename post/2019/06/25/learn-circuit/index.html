模拟电路

电压符号

  VCC：C=circuit 表示电路的意思, 即`接入电路的电压`；
  VDD：D=device 表示器件的意思, 即`器件内部的工作电压`（接电源）；
  VSS：S=series 表示公共连接的意思，通常指电路公共接地端电压(`接地或电源负极`）；
  VEE：E=electron 表示构成物质的基本粒子之一，因带负电，也写作e，通常指负电压供电；
  VDDH：H=high 表示高压，即高压供电端。
  Vpk：信号电压源的峰值
  Vrms：电源的有效值

模拟信号

  1. 在时间上和数值上都是连续的信号成为模拟信号，也就是数学上的连续函数，模拟是一组随时间改变的数据，如某地方的温度变化，车辆在行驶过程中的速度，或电路中某节点的电压幅度等.

  2. 测试信号均为正弦波形,模拟电路中的电压或电流是连续函数形式的模拟信号。可以分为线性电路和非线性电路,模拟电路中，像电阻器、电容器、电感组件及变压器的电流和电压信号呈线性的关系，这类的组件称为线性组件，只由线性组件组成的电路为线性电路。线性电路在分析上比较容易。电流和电压信号不呈线性的组件称为非线性组件，像混频器、调制器、真空管、晶体管放大器、运算放大器及振荡器等，若电路有非线性组件，在分析上比较困难，若电压、电流是在一定数值（称为工作点）附近，可以用小信号模型的方式，将非线性组件用线性组件来模拟，以简化分析和计算。模拟电路的组件也可以分为有源组件及被动组件，有源组件属于非线性组件，像晶体管、真空管及运算放大器等。许多模拟电路的特性都是靠有源组件的特性的产生。

放大电路

  1. 信号的放大是最基本的模拟信号处理功能，通过放大电路实现，放大电路也是构成其他模拟电路，ex:滤波(filter),振荡，稳压等功能电路的基本单元电路

  2. 放大电路的放大是指线性放大，即为放大电路输出信号的信息与输入信号完全一致，不减少也不增加，只改变信号幅度或功率的大小，否则，即为失真

  3. 放大电路为一个双口网络，一个信号输入口和一个信号输出口，一般Us为信号源电压，Rs为信号源内阻，Ri为输入电阻，Ui和Ii分别为输入电压和输入电流，RL为负载电阻，Ro为输出电阻，Uo为输出电压，Io为输出电流

  4. 同相放大电路中输出电压与输入电压相位相同，反向放大电路中输出电压与输入电压相位相反。

  5. 虚短：同相放大电路中Uo=A(Up-Un),由于运放增益A十分大，所以(Up-Un)几乎无穷小,所以理想情况下就有Up=Un。

  6. 虚段：必须在虚短的前提下。由于运放输入电阻Ri无穷大，输出电阻无穷小，所以输入端电流Ip和In基本没有，故认为断路，就认为在输入端没有损耗。当输入端输入一个很小的电流时，输出端也能得到很大电流。

  7. 虚地：反相放大电路中，由于同相端接地（Vp=0），根据虚短所以得到Vp=Vn=0。

    理想电压放大电路输出电阻Ro为0，输入电阻Ri应为无穷大

    电压放大电路适用于信号源内阻Rs较小且负载电阻RL较大的场合

三极管(电流型控制器件)

Bipolar jucntion Transistor：双极性结型晶体管，俗称三极管。在同一个硅片上生成三个杂质半导体区域（NPN或者PNP），两两区域交叠共形成两个PN结。

BJT结构特点： 1. 基区很薄并且掺杂浓度很低（P或N），发射区和集电区是同类型的杂质半导体（两个N或两个P）. 2. 发射区与基区之间的结叫做发射结，集电区与基区之间的结叫做集电结。发射极上的箭头方向一般为发射极加正偏电压是的电流方向。 3. 发射区掺杂浓度高于集电区，集电区面积大于发射区。

电流放大:电流控制电流，三极管是通过流过基极的电流Ib来控制集电极 电流Ic，通过接电阻形式来转换成电压放大。

三极管主要用来控制电流的大小，以共发射极接法为例（信号从基极输入，从集电极输出，发射极接地），当基极电压UB有一个微小的变化时，基极电流Ib也会随之有一小的变化，受基极电流Ib的控制，集电极电流Ic会有一个很大的变化，基极电流Ib越大，集电极电流Ic也越大，反之，基极电流越小，集电极电流也越小，即基极电流Ib控制集电极电流Ic的变化。但是集电极电流的变化比基极电流的变化大得多，这就是三极管的电流放大作用。

NPN

  1. 基极b，发射极e，集电极c。

  2. 截止状态:当Vbe两端电压至少达到0.7V时，三极管才可以达到正常工作状态，否则即为截止状态。三极管工作状态下Vbe两端电压始终保持0.7V。

  3. 放大状态:三极管处于放大状态时，发射结加正向偏置电压，集电结加反向偏置电压（ 简称发射结正偏，发射结反偏）。

    1. Vce > Vbe
    2. Ie = Ib + Ic = （1+β）Ib（说明Ib一定时，Ic与其反偏电压Vce大小无关，只与放大倍数β有关）
    3. Ic = βIb（其他β为三级管放大倍数，电路中放大系数一般为30~80之间）

  4. 饱和状态: 三极管处于饱和状态时，发射结和集电结都加正向偏置电压。

    1. Vce < Vbe
    2. Ie = Ib + Ic
    3. Ic随着Vce增大而显著增加。

场效应管MOS(电压控制性器件，四端元器件)

MOS（metal oxide semiconductor金属氧化物半导体）晶体管是目前半导体产业领域常见的基础元件，一般分P型（n型衬底、p沟道）和N型（p型衬底、n沟道）2种，从引脚上分别是栅极G，源极S，漏极D和衬底B，

MOS是对称结构，Drain和Source

源极的源是指载流子的起点；漏极的漏是指载流子的终点。载流子从源极出发，穿过沟道，到达漏极，从外部看，载流子最终从漏极漏出去了。显然，NMOS和PMOS的载流子是不同的，因此导致了令人困惑的电流方向问题。盯住载流子即可，别被电流方向迷惑。

可以简单地认为，栅极和衬底间的电压超过阈值后，漏极和源极就接通了，而电流大小则是由栅漏源三极间的电压决定。因为MOS是对称结构，所以源极和漏极无区别且可互换。

关于D和S，也就是漏和源，其实是从工艺角度观察的结果。在MOS中，有两种载流子，一种是电子，另一种是空穴，标记为N和P。显然，NMOS的载流子是电子，PMOS的载流子是空穴。NMOS导通后，电子从源极进入沟道，从漏极离开。因为电子的运动方向与电流方向相反，所以电流从NMOS的漏极流向源极。相反地，在PMOS中，虽然空穴也是从源极进入沟道，从漏极离开，但空穴的运动方向与电流方向一致。因此，电流从PMOS的源极流向漏极。

电压控制电流：场效应管是通过栅极电压来控制漏极电流，MOS管只有四个引脚，即漏极、栅极、源极和衬底。

NMOS和PMOS的接法是不一样的

一般情况下MOS管的S和B是直接相连的接输入，D端是输出，G端是控制信号。

NMOS：漏极D→输出信号、栅极G→输入信号、源极S→低电平(或地)、衬底B→低电平(或地)。 PMOS：漏极D→输出信号、栅极G→输入信号、源极S→高电平(或电源)、衬底B→高电平(或电源)。

MOS管有用的管脚就是三个：源极、漏极、栅极。多余的管脚或者是同名管脚（在内部和漏极、栅极相连，特别是有些大功率管），或者是空脚（没有内部连接，只起焊接固定作用）。

光耦

一边是发光二极管，一边是光敏三极管，两边需要不同电源供电，以确保两边器件不受干扰，起到隔离的作用

发生器件

喇叭：无极性器件不分正负极，也是无源蜂鸣器，（不是所有的喇叭都是无极性器件），内部没有振荡源，不能通过直流驱动，只能通过2KHz~5KHz的方波使其发声。

蜂鸣器：有极性，长正段负，有源蜂鸣器，内部含有振荡源，使用1.5V~15V电压即可驱动发声。

继电器

继电器是一种自动开关，通过低电压，小电流去控制高电压，大电流。 特点：有两组电路，一组是低电压控制电路，一组是高电压工作电路。 K代表线圈不部分，另一边代表开关部分。 ## 数码管 LED数码管（LED Segment Displays）由多个发光二极管封装在一起组成“8”字型的器件，引线已在内部连接完成，只需引出它们的各个笔划，公共电极。数码管实际上是由七个发光管组成8字形构成的，加上小数点就是8个。这些段分别由字母a,b,c,d,e,f,g,dp来表示。

共阴极:（CATHODE）负极连一起数码管内部所有的二极管负极连接在一起接到CK端，然后CK端接地GND形成回路，其他单独引出来的正极依次接正电压。

共阳极：（ANODE）正极连一起数码管内部所有的二极管正极连接在一起接到CK端，然后CK端接电源正极，其他单独引出来的负极依次接地GND形成回路。

数字电路
